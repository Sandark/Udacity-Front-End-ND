Best German trains

DB 406

ICE 3, or Intercity-Express 3, is a family of high-speed electric multiple unit trains operated by Deutsche Bahn. It includes classes 403, 406 and 407, which are known as ICE 3, ICE 3M and New ICE 3 respectively.

Stepen J Mason, Flickr / Wikimedia Commons / CC BY-SA 3.0

DB 143

The DR Class 243 is a universal electric locomotive of the Deutsche Reichsbahn which is used for general rail service. Deutsche Bahn lists the locomotive as Class 143. The locomotives of class 143/243 still belong to the most successful class of German electric locomotives.

Thomas Wolf, www.foto-tw.de / Wikimedia Commons / CC BY-SA 3.0

DB 182

The EuroSprinter (also known as Deutsche Bahn Series 182) family of electric locomotives is a modular concept of locomotives for the European market built by Siemens. The internal Siemens product name is ES 64, with ES for EuroSprinter and the number 64 indicating the 6,400 kW power at rail.

Thomas Wolf, www.foto-tw.de / Wikimedia Commons / CC BY-SA 3.0

DB 644

The Talent is a multiple unit railcar manufactured by Bombardier that was developed by Waggonfabrik Talbot in Aachen shortly before the company was acquired by Bombardier in 1995. The name Talent is an acronym in German for TALbot LEichter Nahverkehrs Triebwagen (in English, Talbot light suburban railcar).

Thomas Wolf, www.foto-tw.de / Wikimedia Commons / CC BY-SA 3.0

DB 218

The DB Class 218 (before 1968 the DB Class V 164) are a class of 4-axle, diesel-hydraulic locomotives acquired by the Deutsche Bundesbahn for use on main and secondary lines for both passenger and freight trains. Despite being displaced from many workings by DMUs, electrification, and inherited DR Class 130s, as of 2009 a significant number of the class still remain active throughout Germany

Thomas Wolf, www.foto-tw.de / Wikimedia Commons / CC BY-SA 3.0

DB 103

The Baureihe 103 is a class of electric locomotives in Germany, originally operated by Deutsche Bundesbahn. For a long period, they were perceived as flagships of the DB rolling stock.
